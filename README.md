# MyPlace — site de marque Astro

Site marketing statique premium pour `myplace.ch`.

## Lancer

```bash
cd myplace-brand-site
npm install
npm run dev
```

Astro est volontairement en v4 pour fonctionner avec Node 20.

## Build

```bash
npm run build
npm run preview
```

## Structure
- `/` home premium
- `/contact` page contact façon ticketing
- pages SEO métier : régie, architecte, courtier, PPE
- pages SEO produit : portail client, gestion documentaire, support client, tarifs, fonctionnalités

## Déploiement
- `myplace.ch` : ce site Astro
- `app.myplace.ch` : app Next.js
